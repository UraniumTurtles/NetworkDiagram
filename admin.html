Client Information


                

                

                    Step-by-step Instructions:



                    Step 1: Create the client file

                    • Create a new file: data/clients/[client-id].js

                    • Copy the generated code above and paste it into this new file

                    • Save the file



                    Step 2: Import the client in networkData.js

                    • Open data/networkData.js

                    • Add this import at the top with the other imports:

                    import [clientName] from './clients/[client-id].js';
                    • Add the client to the clientsMap object:

                    '[client-id]': [clientName],


                    Step 3: Add to the area

                    • Open data/areas.js

                    • Find the area object for "[area]"

                    • Add '[client-id]' to the clientIds array



                    Step 4: Test

                    • Open index.html in your browser

                    • Navigate to the area and verify the new client appears

                    • If you see errors, check the browser console (F12)

Enter VMs as a JSON array. Each VM should have name, role, os, and ip fields.

                

            


            

                
                
            


            

                

                    
                    

                

                

                    Step-by-step Instructions:



                    Step 1: Find the location file

                    • Navigate to data/clients/ and open the client file that contains the location you want to edit

                    • Find the location object within the locations array



                    Step 2: Add the device code

                    • Inside the location's diagram object, find the [device-type] array

                    • If the array doesn't exist, create it: [device-type]: []

                    • Copy the generated device code above and paste it into the array

                    • If there are existing devices, add a comma after the previous device



                    Example:

                    diagram: {
    size: { width: 960, height: 560 },
    [device-type]: [
        // ... existing devices ...
        // Paste your new device here
    ],
    links: [...]
}


                    Step 3: (Optional) Add connections

                    • If this device connects to other devices, add link objects to the links array

                    • Example: { from: { type: '[device-type]', id: '[device-id]' }, to: { type: 'switches', id: 'switch-1' } }



                    Step 4: Test

                    • Open index.html in your browser

                    • Navigate to the location and verify the new device appears in the correct position

                    • If you see errors, check the browser console (F12)
